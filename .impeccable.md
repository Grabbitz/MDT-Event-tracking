## Design Context

### Users
The Modern Trade (MDT) team. They use this tool to track event calendars, sales targets, and plan campaigns. The context is professional, data-centric, but workspace-oriented.

### Brand Personality
- Clean
- Precise
- Professional

3-word personality: **Clean, Clinical, Workspace**

### Aesthetic Direction
- **Visual Tone**: Apple-style minimalism. "Clean clinic" feel.
- **Theme**: Light mode primarily, using pure whites and subtle neutrals.
- **Typography**: Maintaining existing Noto Sans Thai and Aptos, but with refined hierarchy and better use of weight (avoiding over-bolding).

### Design Principles
1. **Generous White Space**: Every element must have room to breathe. Avoid cramped layouts.
2. **Subtle Hierarchy**: Use space and font weight before color or borders to distinguish sections.
3. **Soft Geometry**: Large rounded corners (min 12px-16px) and subtle, almost invisible borders.
4. **Data Clarity**: MDT data (events, targets) should be the hero. UI chrome should recede into the background.
5. **Functional Minimalism**: No decorative elements that don't serve a purpose (e.g., no unnecessary background gradients).
